# Wumbo Brand Guidelines

**Version**: 1.0  
**Last Updated**: December 2024

---

## Brand Overview

**Name**: Wumbo  
**Tagline**: Making Your Money Bigger  
**Domain**: wumbo.app  
**Mission**: Empower families to grow their wealth through simple, automated budgeting

---

## Brand Personality

### Core Attributes

- **Approachable** - Finance doesn't have to be intimidating  
- **Growth-Oriented** - Everything points upward  
- **Trustworthy** - Playful name, serious security  
- **Empowering** - You're in control, we're the guide  
- **Smart** - Intelligent automation, wise decisions

### Brand Voice

**Tone**: Friendly financial advisor, not stuffy banker  
**Language**: Clear and conversational, never condescending  
**Humor**: Light and optimistic, not sarcastic  

**Do's**:
- "Let's grow your savings together"
- "Your money is getting bigger"
- "Small changes, wumbo results"

**Don'ts**:
- Financial jargon without explanation
- Shame or guilt about spending
- Overly silly/immature language

## Visual Identity

### Logo Concepts

#### Primary Logo Direction: "The Upward W"

**Concept 1: Growth Arrow W**
```
  ↑  ↑
 / \/ \
/      \
```
- W shape with upward-pointing peaks
- Suggests growth and expansion
- Clean, geometric, modern
- Works as icon or full wordmark

**Concept 2: Expanding W**
- W that appears to be growing/getting bigger
- Animated version could expand on hover
- Reinforces "wumbo = bigger"

**Concept 3: W + Wombat (Optional Mascot)**
- Minimal wombat silhouette integrated into W
- Not cartoonish - sophisticated like Duolingo owl
- Optional, not required in all uses

#### Logo Variations Required

- **Primary**: Full color wordmark + icon
- **Icon Only**: For app icon, favicon  
- **Monochrome**: Black and white versions
- **Minimum Size**: Readable at 16px height

### Logo Usage

**Clear Space**: Minimum space around logo = height of "W" letterform

**Minimum Sizes**:
- Digital: 24px height (icon), 120px width (wordmark)
- Print: 0.25 inches height

**Don't**:
- ❌ Rotate or skew the logo
- ❌ Change colors outside approved palette
- ❌ Add effects (drop shadow, glow)
- ❌ Place on busy/low-contrast backgrounds
- ❌ Stretch or distort proportions

- ✅ Wumbo Green button for "Add to Savings Goal"

- ❌ Using red for a category that's not negative/expense

- ❌ Too many colors on one screen (limit to 3-4)
- ❌ Purely decorative use of Warning Orange or Error Red

## Typography

### Font Families

#### Primary: Inter
**Modern, highly readable, excellent for UI, free/open-source**

**Where to use**: All interface text, headings, body copy, buttons

**Weights**:
- Regular (400) - Body text
- Medium (500) - Emphasized text, labels
- Semibold (600) - Subheadings, buttons
- Bold (700) - Headings

**Fallback**:
```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', 
             'Roboto', 'Oxygen', 'Ubuntu', sans-serif;
```

#### Monospace: JetBrains Mono
**Excellent for numbers/currency alignment, modern aesthetic**

**Where to use**: Currency amounts, account numbers, data tables

**Weights**: 
- Regular (400)
- Medium (500)

**Fallback**:
```css
font-family: 'JetBrains Mono', 'SF Mono', 'Consolas', 
             'Monaco', monospace;
```

### Type Scale

Base: 16px on all devices

| Name | Size | Usage |
|------|------|-------|
| xs | 12px | Tiny labels, captions |
| sm | 14px | Small text, metadata |
| base | 16px | Body text (DEFAULT) |
| lg | 18px | Large body, emphasized |
| xl | 20px | H4 |
| 2xl | 24px | H3 |
| 3xl | 30px | H2 |
| 4xl | 36px | H1 |
| 5xl | 48px | Display/Marketing |

### Currency Formatting

**Standard**:
```
$1,234.56
```
- Font: JetBrains Mono
- Always 2 decimal places
- Comma separators
- No space after $

**Large Numbers**:
```
$1.2K  or  $1,234
```
- Abbreviated when space-constrained
- Context-dependent

**Colorized**:
- Income: Wumbo Green `#10B981`
- Expense: Error Red `#EF4444`
- Neutral: Gray 500 `#6B7280`
- Optional +/- prefix

**Base Unit**: 4px (Tailwind standard)

| Token | Value | Common Usage |
|-------|-------|--------------|
| 1 | 4px | Tiny gaps |
| 2 | 8px | Compact spacing |
| 3 | 12px | Button/input padding (vertical) |
| 4 | 16px | Button/input padding (horizontal), default gap |
| 6 | 24px | Card padding |
| 8 | 32px | Section spacing |
| 12 | 48px | Large section gaps |
| 16 | 64px | Major page sections |

**Common Patterns**:
- Button: `px-4 py-2` (16px × 8px)
- Input: `px-3 py-2` (12px × 8px)
- Card: `p-6` (24px all sides)
- Page: `p-4` mobile, `p-8` desktop

#### Sizes

| Size | Padding | Text | Usage |
|------|---------|------|-------|
| sm | `px-3 py-1.5` | 14px | Compact spaces |
| md | `px-4 py-2` | 16px | DEFAULT |
| lg | `px-6 py-3` | 18px | Emphasis |

#### States

- **Disabled**: 50% opacity, cursor-not-allowed
- **Loading**: Spinner, disabled
- **Focus**: 2px ring in brand color

### Input Fields

**Base Style**:
- Border: 1px solid Gray 200 `#E5E7EB`
- Radius: 8px
- Padding: `px-3 py-2`

**States**:
- **Default**: Gray border
- **Focus**: Blue border + ring
- **Error**: Red border + ring
- **Disabled**: Gray background, lighter text

**Currency Input** (Special):
- Font: JetBrains Mono
- Leading $ icon
- Right-aligned
- Format on blur: commas, 2 decimals

### Cards

**Base Style**:
- Background: White (light) / Slate 800 (dark)
- Border: 1px solid Gray 200 (optional)
- Radius: 16px
- Padding: 24px
- Shadow: sm (md on hover if interactive)

**Card Types**:

**Transaction Card**:
- Icon left (category)
- Amount right (colored, monospace)
- Minimal border
- Hover: background shift

**Budget Card**:
- Progress bar (colored by %)
- Category icon
- Clear hierarchy

**Goal Card**:
- Progress visualization
- Target date
- Motivational copy

### Progress Bars

**Style**:
- Height: 8px (thin) or 12px (standard)
- Radius: Full (pill)
- Background: Gray 200
- Smooth transition

**Colors by Status**:
- < 80%: Wumbo Green
- 80-95%: Warning Orange
- \> 95%: Error Red

### Badges

**Style**:
- Radius: Full (pill)
- Padding: `px-2.5 py-0.5`
- Font: 12px, medium weight

**Variants**:
- Default: Gray
- Success: Light green bg, dark green text
- Warning: Light orange bg, dark text
- Error: Red bg, white text
- Info: Light blue bg, dark blue text

### Modals

**Structure**:
- Overlay: Black 40% opacity
- Content: White card, 24px radius
- Shadow: xl
- Max width: 500px (mobile: full - 32px)
- Padding: 24px

**Actions**:
- Primary: Wumbo Green button (right)
- Cancel: Secondary button (left)

## Animation & Motion

### Principles

- **Purpose**: Motion has meaning
- **Speed**: Fast but noticeable
- **Easing**: Natural curves (ease-in-out)

### Timing

| Speed | Duration | Usage |
|-------|----------|-------|
| fast | 150ms | Hover, focus |
| DEFAULT | 200ms | Standard transitions |
| slow | 300ms | Modals, page transitions |
| slower | 500ms | Complex animations |

### Common Animations

**Hover**: 150ms, scale to 102% or background shift

**Page Transition**: 200ms fade + slight upward slide (16px)

**Modal**: 300ms scale (95% → 100%) + fade

**Success**: 500ms check mark draw + green pulse

**Loading**: Skeleton pulse (1.5s loop) or rotating spinner (1s)

**Microinteractions**:
- Progress bar: Smooth width transition
- Currency: Count-up animation
- New item: Slide in from top

## Accessibility (WCAG 2.1 AA)

### Color Contrast

**Requirements**:
- Normal text: 4.5:1 minimum
- Large text (≥18px): 3:1 minimum
- UI components: 3:1 minimum

**Verified**:
- ✅ Gray 600 on white: 8.6:1
- ✅ White on Wumbo Green: 4.1:1
- ⚠️ Wumbo Green on white: 3.4:1 (large text only)

### Keyboard Navigation

- Visible focus ring (2px, 2px offset)
- Logical tab order
- Skip links
- Modal focus trap

### Screen Readers

- Semantic HTML
- Proper heading hierarchy
- Labels on all inputs
- ARIA labels where needed
- `aria-live` for dynamic content

### Touch Targets

- Minimum: 44×44px (iOS), 48×48px (Android)
- Spacing: 8px between targets

## Summary: The Wumbo Look

**In one sentence**: Clean, growth-oriented design with Wumbo Green as the hero color, balanced by trustworthy blue and professional grays.

**The Essence**:
1. **Color**: Green (growth) is the star
2. **Type**: Inter (UI) + JetBrains Mono (money)
3. **Shape**: Rounded, upward-pointing
4. **Motion**: Smooth and purposeful
5. **Voice**: Encouraging financial friend

**Brand Promise**: Making your money bigger shouldn't be complicated - it should be clear, automated, and empowering.
